PeopleSoft-Build表注意事项

发表于 2020-09-16 更新于 2021-06-19 分类于 学习 本文字数： 171 阅读时长 ≈ 1 分钟
在PeopleTool开发工具中build表的时候需要注意下列规则...

delete字段不能迁移

迁移工程时，如果删除了表中的某些字段，虽然对比能够检视出变化的表，但是不会自动勾选，需要手动在upgrade中勾选delete的字段，才可以迁移。

字段长度改短或者删除已经存有值的字段；

build的时候会报error，因为alter 的时候会直接跳过；这个时候需要调整一下alter的设置，忽略其长度或者及时有值也要删除；
